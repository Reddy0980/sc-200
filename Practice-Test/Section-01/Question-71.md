Question Type : Multiple Single Choice

## Question
A security analyst wants to update an existing analytics rule in Microsoft Sentinel. Can the analyst perform the following tasks to update the rule? 

## Labels
Label 1 : Change the query used by the rule 
Label 2 : Modify the schedule of the rule
Label 3 : Change the data sources used by the rule 

## Options
Option 1 : Yes
Option 2 : No

## Answers
Label 1 : Option 1 : 4
Label 2 : Option 1 : 5
Label 3 : Option 1 : 4

## Reference Links
https://learn.microsoft.com/en-us/azure/sentinel/manage-analytics-rule-templates 

## Explanation
A security analyst can perform all three tasks to update an existing analytics rule in Microsoft Sentinel. Changing the query used by the rule allows the analyst allows the analyst to refine the criteria for identifying suspicious behavior. Modifying the rule schedule allows the analyst to adjust the frequency with which the rule is run. Changing the data sources used by the rule allows the analyst to adjust the sources of information used to detect suspicious behavior.
